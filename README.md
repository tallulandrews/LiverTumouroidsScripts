# Liver Cancer Organoid Analysis Pipeline

## Purpose
This repository holds custom scripts and piplines used to analyze the scRNAseq data in "Dietary unsaturated fatty acids reprogram dormant human cancer cells into tumour-initiating cell states". This includes in-house Smartseq2 tumour organoid data, and matched publicly available primary tumour 10X genomics datasets.

## Installation
This scripts are not a package, rather a bunch of in-house scripts and data-files for a one-off custom analysis. To obtain a copy you can download individual scripts or clone the whole reposity.
